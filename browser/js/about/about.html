About Us

Here at BPS our goal is to help you make fun, exciting, colorful HTML pages without you needing to touch HTML or CSS code at all. Our editor is an easy drag and drop interface with many tools to help you make any website that you want. The projects you build will be styled with Materalize CSS, making your webpages look more modern without any extra work.



How To Use Our Site

Using our site is easy! To start a new project, simply go to your Projects page and click the New Project button. It will take you straight to the editor page where you can enter a name for your project.

From there, clicking the button like the one below will open up a sidebar with our project tools.
