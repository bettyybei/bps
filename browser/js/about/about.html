About Us

Here at Paletro our goal is to help you make fun, exciting, colorful HTML pages without you needing to touch HTML or CSS code at all. Our editor is an easy drag and drop interface with many tools to help you make any website that you want. The projects you build will be styled with Materalize CSS, making your webpages look more modern without any extra work.

Paletro means 'palette' in Esperanto, a language created to be easy to learn for people all around the world. We hope that Paletro will be as easy to use, helping you create the webpages you always wanted.



How To Use Our Site

We put together a little guide for when you get stuck on creating your projects.

  • filter_dramaCreate a new project

    To start a new project, simply go to your Projects page and click the New Project button.
    It will take you straight to the editor page where you can enter a name for your project.

  • createEditor Page
    label_outline
    Opens sidebar
    save
    Saves Project
    color_lens
    Select Color
    iso
    Select Shade
    border_all
    Toggles Grid

    Project Details is where you can change your project name anytime.
    Background Color is where you can change the background color to your page.
    New Components is where you add items to your webpage. We currently have: Navbars, Logos, Buttons, Divs, Text Boxes, and Headers
    New Image is where you can add an image. Simply paste an image url you find online and you can use it on your site.

    Don't forget to save your project!

  • file_uploadExporting your webpage

    After you're finished, export your webpage from your Projects page. From there you have 2 options:
    1. Download a zip file with all of your project files
    2. Copy all of the code to your clipboard and paste it as the HTML for your site
