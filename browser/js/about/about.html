About Us

Here at Paletro our goal is to help you make fun, exciting, colorful HTML pages without you needing to touch HTML or CSS code at all. Our editor is an easy drag and drop interface with many tools to help you make any website that you want. The projects you build will be styled with Google's Material Design, making your webpages look more modern without any extra work.

Paletro means 'palette' in Esperanto, a language created to be easy to learn for people all around the world. We hope that Paletro will help you communicate with the HTML and CSS world with our fun editor, bridging the language gap between graphic designers and computer programmers.



How To Use Our Site

We put together a little guide for when you get stuck on creating your projects.

  • filter_dramaCreating a new project

    To start a new project, simply go to your Projects page and click the New Project button.
    From there you can name the project and add new pages.

  • createNavigating the Editor Page
    label_outline
    Opens sidebar
    save
    Saves Project
    folder_open
    Returns to Project
    color_lens
    Select Color
    iso
    Select Shade
    border_all
    Toggles Grid
    delete_forever
    Drag Here to Delete

    Project Details is where you can change your project name anytime.
    Background Color is where you can change the background color to your page.
    New Components is where you add items to your webpage. We currently have: Navbars, Logos, Buttons, Divs, Text Boxes, and Headers
    New Image is where you can add an image. Simply paste an image url you find online and you can use it on your site.

    Select your component to change its color! If it has blue dotted lines around it, that means that it is selected.
    Double-click on headers and textboxes to change their text colors.

    Don't forget to save your project!

  • file_uploadExporting your webpage

    After you're finished, export your webpage from your Projects page. From there you have 2 options:
    1. Download a zip file with all HTML pages for the project
    2. Copy the code to your clipboard and paste it as the HTML for your site
